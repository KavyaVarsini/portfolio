# Personal Portfolio | Kavya Varsini S

Welcome to the repository of my personal computer science portfolio. This website is a premium, modern showcase of my projects, skills, certifications, and research contributions as a **Full Stack Developer** and **AI Enthusiast**.

The design system features a dark luxury layout inspired by Vercel, Stripe, and Linear. It utilizes glassmorphism, soft glows, and clean grid spacing, avoiding flashy animations and colorful gradients to maintain a premium developer aesthetic.

---

## ⚡ Tech Stack

- **Framework**: React 19 & TypeScript
- **Styling**: Tailwind CSS v4 (Direct CSS compiler plugin)
- **Animations**: Framer Motion 12
- **Icons**: Lucide Icons
- **Build Tool**: Vite 8
- **Target Deployment**: Vercel

---

## 🚀 Key Features

- **Linear-style Navbar**: Glassmorphic floating header navigation with viewport tracking active states.
- **Rotating Roles Intro**: Premium hero entry highlighting core competencies with floating micro-particles.
- **AvatarSpeak Component**: Interactive speech avatar supporting drag-and-drop, clicks, and clipboard pasting of custom image headshots, complete with concentric mouth-pulse ripples.
- **Academic Timeline**: Displays B.Sc Computer Science with Data Analytics history (9.0 CGPA at KPR College).
- **Featured Projects**: Interactive card grid highlighting *ShopSphere* (with live demo), *Centralized Power Loom shift monitor*, and *Femmora* with gradient-border hover elevations.
- **Interactive Contact Portal**: Connects directly to my email address (`kavyasenthil505@gmail.com`) using FormSubmit AJAX.

## 🛠️ Local Development

To run this project locally, navigate into the directory and execute the developer scripts:

```bash
# Install dependencies
npm install

# Start local development server
npm run dev

# Compile production-ready assets
npm run build
```
